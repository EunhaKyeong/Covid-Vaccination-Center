# 코로나19 예방접종센터 지도 서비스

## 📹 기능 소개 영상
- 공공데이터 & NAVER MAP을 활용한 코로나19 예방접종센터 목록 조회  
- 현재 위치 기능을 통한 인접 예방접종센터 목록 조회  
- 코로나19 예방접종센터 상세 정보 조회  

## 🛠️ 활용 기술
- Language: Kotlin  
- Design Pattern: MVVM
- UI Layout: XML
- Network: Retrofit2
- Jetpack: View Bidning, DataBinding, ROOM
- Naver MAP
- DI: Hilt
- Coroutine Flow
